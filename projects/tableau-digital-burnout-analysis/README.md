# Digital Burnout Analysis – Tableau Dashboard

This project analyzes patterns of digital burnout using interactive data visualizations built in Tableau. The dashboard explores how screen time, work patterns, and digital engagement contribute to burnout levels among individuals.

## Project Overview

The goal of this project is to identify behavioral and work-related factors that contribute to digital burnout and present these insights through an interactive Tableau dashboard.

The dashboard enables users to explore burnout trends across different demographics, work environments, and technology usage patterns.

## Key Insights

- Relationship between screen time and burnout levels
- Impact of work hours and remote work on digital fatigue
- Burnout distribution across age groups and professions
- Trends in digital device usage

## Dashboard Features

- Interactive filters and slicers
- Visual breakdown of burnout levels
- Comparative analysis across demographic groups
- Dynamic charts and dashboards
