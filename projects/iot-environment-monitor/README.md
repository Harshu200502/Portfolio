# IoT Environmental Monitoring System

An IoT-based monitoring system designed to collect environmental data using sensors connected to a Raspberry Pi.

## Overview
The system captures real-time environmental data such as temperature, humidity, and motion, and uploads it to cloud platforms for monitoring and analysis.

## Key Features
- Temperature and humidity monitoring
- Motion detection using PIR sensor
- Real-time data logging
- Cloud data integration

## Technologies
- Raspberry Pi
- MicroPython
- IoT Sensors
- Cloud data platforms

## Applications
- Smart monitoring systems
- Environmental analytics
- IoT-based data collection
