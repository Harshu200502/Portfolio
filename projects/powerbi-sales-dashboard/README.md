# Sales Data Analytics Dashboard – Power BI

This project presents a data analytics dashboard built using Power BI to analyze sales performance, customer behavior, and revenue trends.

## Project Overview

The dashboard provides interactive visualizations that help businesses understand sales patterns and identify key performance indicators for better decision making.

## Key Insights

- Revenue trends over time
- Top performing products
- Regional sales distribution
- Customer segmentation
- Monthly performance metrics

## Tools & Technologies

- Power BI
- Data Visualization
- Data Cleaning
- Business Intelligence

## Dashboard Features

- Interactive filters and slicers
- KPI indicators
- Dynamic charts and graphs
- Business performance insights
